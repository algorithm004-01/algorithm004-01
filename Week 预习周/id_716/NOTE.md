# NOTE

### 预习周

#### 学习方法

- 人脑适合记忆树状结构的知识，不适合零散的知识点，将知识点串起来有助于理解和记忆
- 如果要达到专业水平，顶尖水平，需要做刻意练习，哪里不熟练就找准之后做刻意练习，形成肌肉记忆

##### 关于刻意练习

- 最大误区：做算法题，只做一遍是不够的，要反复练习
- 刷题方法：5遍刷题法，过遍数
- 刻意练习弱项，刻意练习不舒服的地方，跳出舒适区
- 刻意练习要练习基本功
- 即使获得反馈，主动学习别人的代码

#### 关于职业训练

- 拆分知识点
- 刻意练习
- 反馈

#### 算法和数据结构的串联

使用脑图将算法和数据结构的核心知识点串联起来，在整体上有一个全貌。自己手动绘制脑图，加入自己的理解。
